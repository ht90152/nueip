1.extend.java & interface.java<br>
繼承:<br>
可以定義方法、變數、常數,可使用父類別的所有定義。<br>
可複寫父類別的方法。<br>
只能繼承(extends)1個父類別。<br>
介面:<br>
只能定義全域常數、抽象方法。<br>
一定要實做interface內抽象方法。<br>
可實做(implements)多個interface<br>
